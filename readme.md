# Hedera Go Template

This repository serves as an example of a Hedera Golang project.

## Version

This project is compatible with Go v2

## Clone the project

```
$ git clone https://github.com/pathornteng/hedera-go-template.git
$ cd hedera-go-template
```

## Run the project

```
go run .
```

## Build the project

```
go build .
```
